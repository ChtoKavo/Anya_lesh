# 🚀 БЫСТРЫЙ СТАРТ: АДМИН ПАНЕЛЬ

## За 5 минут до работающей админ панели!

### 1️⃣ Применить миграцию БД (1 минута)

```bash
# Подключиться к MySQL
mysql -u root -p

# Выбрать БД
USE mydiplom;

# Копировать и выполнить весь SQL из файла:
# mydiplom/database/migration_admin_features.sql
```

**ИЛИ через командную строку:**
```bash
mysql -u root -p mydiplom < mydiplom/database/migration_admin_features.sql
```

### 2️⃣ Создать первого админа (1 минута)

В MySQL выполнить:
```sql
-- Найдите пользователя по email и дайте ему админ роль
UPDATE users SET role = 'admin' WHERE email = 'your_email@example.com';

-- Проверить
SELECT id, email, role FROM users WHERE email = 'your_email@example.com';
```

### 3️⃣ Перезагрузить серверы (2 минуты)

**Backend:**
```bash
cd mydiplom-backend
npm start
```

**Frontend (в новом терминале):**
```bash
cd mydiplom
npm run dev
```

### 4️⃣ Протестировать! (1 минута)

1. Открыть браузер: http://localhost:5173
2. Логинуться как админ
3. Посмотреть в хэдере ссылку **"🔐 Админ Панель"**
4. Кликнуть и наслаждаться! 🎉

## 🎯 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Как пользователь написать в поддержку?
1. Профиль → Поддержка → "Написать поддержку"
2. Заполнить форму
3. Отправить
4. Получить ответ от админа

### Как администратор ответить?
1. Админ панель → Поддержка
2. Найти запрос
3. Нажать "Ответить"
4. Написать ответ
5. Отправить

### Как посмотреть рейтинг?
1. Админ панель → Рейтинг
2. Видеть топ пользователей
3. Разворачивать полный список

## ❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

**Q: Админ панель не показывается?**
A: Проверьте, что в БД role = 'admin' для вашего пользователя:
```sql
SELECT * FROM users WHERE email = 'your_email@example.com';
```

**Q: Как сделать пользователя админом?**
A: 
```sql
UPDATE users SET role = 'admin' WHERE id = 1;
```

**Q: Как отключить админа?**
A:
```sql
UPDATE users SET role = 'user' WHERE id = 1;
```

**Q: Где искать ошибки?**
A: 
- Браузер: F12 → Console
- Backend: Консоль терминала

**Q: Можно ли создать нескольких админов?**
A: Да! Просто дайте роль admin нескольким пользователям.
